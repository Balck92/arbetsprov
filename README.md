Install rss-parser:
  npm i rss-parser
  
Install prop-types
  npm i prop-types

Install toastify
  npm i toastify
  
Install node-sass
  npm i node-sass

Run in webbrowser (localhost:3000):
  
  cd /arbetsprov
  npm start 
  
  
All React-components were locally imported after creating the react app (npm init react-app arbetsprov).
